## ğŸš€ Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites

*   **Node.js** (v18 or higher)
*   **npm** or **yarn**

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/your-username/leacus-store.git
    cd leacus-store
    ```

2.  **Install dependencies**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    # or
    yarn dev
    ```

4.  **Open your browser**
    Navigate to [http://localhost:3000](http://localhost:3000) to see the application running.

## ğŸ¨ Design Philosophy

At *Leacus.*, we believe in **"Less is More"**. Our design philosophy revolves around:

*   **Whitespace**: Generous use of whitespace to create a sense of luxury and focus.
*   **Typography**: A sophisticated pairing of *Playfair Display* (Serif) for headings and *Inter* (Sans-serif) for body text.
*   **Micro-interactions**: Subtle hover effects and smooth transitions to enhance user engagement without being distracting.
